# AI Engineering Journey 🚀

A 6-month hands-on journey to become an AI Engineer and freelancer.
Built in public — no courses, just daily projects.

---

## 🛠️ Projects Built

### Day 1 — First AI API Call
- Connected to Google Gemini API using Python
- Made live API calls and got AI-generated responses
- **Tech:** Python, Gemini API, python-dotenv

### Day 2 — AI Business Name Generator
- Built a function-based tool that generates culturally relevant Pakistani business names
- Used prompt engineering to control AI output style and format
- **Tech:** Python, Gemini API, F-strings, Functions

### Day 3 — Conversational AI Chatbot (Ustad Saab)
- Built a stateful chatbot with persistent conversation memory
- Designed a full personality via system prompts
- **Tech:** Python, Gemini API, Conversation History, While loops

### Day 4 — Web Scraper + AI Proposal Generator
- Built a tool that reads any business website and generates a personalized sales proposal
- Scraped real Pakistani brand Sapphire and generated a specific AI automation proposal
- **Tech:** Python, BeautifulSoup, Requests, Gemini API

### Day 5 — CLI Proposal Tool
- Upgraded the proposal generator into a proper command line tool
- Added argument parsing, timestamps, and organized file output
- **Tech:** Python, sys.argv, os, datetime
